# Descargo / Legal

## 1) Sitio web independiente (sin afiliación)

Este sitio web es un **recurso informativo independiente** creado para análisis, crítica y discusión de interés público.

No está **afiliado, respaldado, patrocinado ni operado** por Nueva Acrópolis o cualquiera de sus representantes, ramas o entidades relacionadas.

Si busca información oficial, utilice los sitios web oficiales de la organización y los canales de comunicación.

## 2) Uso de nombres y marcas (sólo identificación)

Cualquier referencia en este sitio a “**Nueva Acrópolis**” y nombres relacionados, términos, logotipos o marcas comerciales se hacen **únicamente con el fin de identificar la organización que se está tratando** y para ofrecer comentarios.

Todas las marcas, marcas de servicio y logotipos son propiedad de sus respectivos propietarios. Este sitio web **no reclama ninguna propiedad** de marcas de terceros y no las utiliza para implicar patrocinio, asociación o respaldo.

## 3) No suplantación / no representación oficial

No nos presentamos como Nueva Acrópolis, ni como representantes, empleados, agentes o socios autorizados de Nueva Acrópolis.

El diseño, la maquetación y el contenido de este sitio web se crean de forma independiente y están destinados a evitar la confusión con cualquier recurso oficial.

## 4) Objetivo y limitaciones del contenido

El contenido de este sitio se proporciona **con fines informativos y educativos**. Refleja el análisis e interpretación de:

- fuentes disponibles públicamente,
- documentos citados,
- testimonios o declaraciones que se nos han proporcionado,
- y otros materiales utilizados en el contexto del comentario.

Si bien procuramos ser precisos y justos, no podemos garantizar que toda la información sea completa, actual o esté libre de errores. Nada en este sitio debe entenderse como asesoramiento jurídico profesional.

## 5) Opiniones y declaraciones

A menos que se indique explícitamente como una cita directa de una fuente, partes de este sitio web pueden contener **opiniones, análisis y comentarios**. Dicho contenido se proporciona de buena fe y está destinado a la discusión pública.

## 6) Enlaces externos

Este sitio web puede incluir enlaces a sitios web de terceros para referencia. No controlamos esos sitios web y no somos responsables de su contenido, disponibilidad o políticas.

## 7) Derechos de autor y uso justo / cita

Respetamos los derechos de propiedad intelectual. Cuando se utilizan materiales de terceros, se emplean con fines como crítica, comentario, investigación e identificación, y suelen ir acompañados de contexto y/o referencias.

Si cree que cualquier contenido de este sitio web infringe sus derechos de autor u otros derechos, póngase en contacto con nosotros con detalles específicos (véase “Contacto” a continuación). Revisaremos las solicitudes fundamentadas.

## 8) Correcciones, derecho de respuesta y solicitudes de retirada

Tomamos la precisión en serio. Si usted cree que cualquier contenido en este sitio web es inexacto, engañoso o publicado ilegalmente, por favor envíe un mensaje incluyendo:

- la URL(s) exacta de la(s) página(s) en cuestión,
- las declaraciones específicas que usted disputa,
- una explicación clara de por qué cree que es inexacto o ilegal,
- y pruebas justificativas (documentos, enlaces u otras fuentes verificables).

Revisaremos su solicitud y, cuando proceda, publicaremos una corrección, aclaración o actualización del contenido. Las solicitudes abusivas, vagas o no respaldadas pueden no recibir respuesta.
