## บทสรุป

เมื่อสิ้นสุดการดำเนินงานของตน คณะกรรมาธิการจำเป็นต้องยอมรับว่า ปรากฏการณ์ลัทธินั้นน่าจะฝังตัวอยู่ในสังคมของเราไปอีกยาวนาน ปรากฏการณ์นี้สะท้อนถึงความบกพร่องที่หลีกเลี่ยงไม่ได้ของสังคม และบางครั้งก็ทำหน้าที่เป็นวาล์วนิรภัยที่ช่วยให้ความเป็นชายขอบซึ่งไม่อาจกดทับได้มีทางระบาย สิทธิในการแตกต่างก็ต้องได้รับการคุ้มครองเช่นกัน แต่อย่างไรก็ตาม เราไม่อาจยอมให้การใช้สิทธินี้กลายเป็นภัยคุกคามต่อหลักการบางประการ โดยเฉพาะอย่างยิ่งสิทธิในการคุ้มครองผู้ที่อ่อนแอกว่า ด้วยเหตุนี้ การต่อสู้กับความเบี่ยงเบนในเชิงลัทธิจึงยังคงเป็นภารกิจที่ในสายตาของคณะกรรมาธิการถือเป็นข้อเรียกร้องสำคัญ ซึ่งต้องอาศัยเครื่องมือที่เหมาะสมยิ่งขึ้น ความพยายามครั้งใหม่ในการสร้างความตระหนักรู้ในสังคม และการตื่นตัวในระดับนานาชาติ

คณะกรรมาธิการเชื่อมั่นว่าความขัดแย้งที่มักถูกหยิบยกขึ้นมาเสมอระหว่างเสรีภาพแห่งมโนธรรมกับความสงบเรียบร้อยของสาธารณชน ในเรื่องการกระทำของลัทธินั้น แท้จริงแล้วไม่อาจใช้การได้เลย เป้าหมายไม่ใช่การตั้งหลักการทั้งสองนี้ให้เผชิญหน้ากัน แต่ตรงกันข้าม คือการทำให้เห็นถึงความเกื้อกูลซึ่งกันและกัน การคุ้มครองเสรีภาพแห่งมโนธรรมไม่อาจตั้งอยู่บนพื้นฐานของการละเลยเสรีภาพส่วนบุคคลที่สำคัญยิ่ง ได้แก่ เสรีภาพในการไปมาค้าขาย การถือครองและใช้ทรัพย์สิน การดูแลรักษาร่างกายและสุขภาพของตน การได้รับการคุ้มครองจากการใช้อำนาจในทางที่ผิด การปกป้องตนเองจากการละเมิดต่อผลประโยชน์ทั้งด้านศีลธรรมและด้านวัตถุของตนเอง … โดยการทำให้การโจมตีเสรีภาพเหล่านี้จากความเบี่ยงเบนแบบลัทธิถูกติดตามและปราบปรามได้อย่างมีประสิทธิภาพยิ่งขึ้น คณะกรรมาธิการตั้งใจให้เสรีภาพแห่งมโนธรรมและการแสดงความคิดเห็น ซึ่งอยู่ในหัวใจของการเคารพสิทธิมนุษยชน ได้รับการค้ำประกัน

(หน้า 254)

## สรุปข้อเสนอของคณะกรรมาธิการ
(ตัวเลขอ้างอิงถึงเลขหน้าของรายงาน)

### 1. การปรับปรุงสถานะทางกฎหมายของสมาคม

- เสริมสร้างความเป็นประชาธิปไตยภายในสมาคม: โดยกำหนดให้สมาคมที่มีงบประมาณเกินเกณฑ์หนึ่ง ซึ่งอาจกำหนดไว้ที่ 500.000 ฟรังก์ ต้องจัดให้มีการประชุมใหญ่สามัญประจำปีอย่างน้อยปีละครั้ง และจัดทำรายงานเชิงจริยธรรมและรายงานการเงินเสนอต่อสมาชิก (หน้า 51 และ 52) ;
- กำหนดให้โครงสร้างสมาคมที่มีน้ำหนักทางเศรษฐกิจระดับหนึ่ง — เช่น สมาคมที่มีงบประมาณประจำปีเกิน 500.000 ฟรังก์ — ต้องปฏิบัติตามข้อผูกพันขั้นต่ำด้านการเปิดเผยข้อมูล (งบกำไรขาดทุนและงบดุล) (หน้า 51 และ 52) ;
- ใช้ประโยชน์จากการเลื่อนการบังคับใช้คำสั่งลงวันที่ 15 กันยายน 1998 ว่าด้วยภาษีของสมาคม เพื่อคำนึงถึงความเสี่ยงของการใช้กฎใหม่ในลักษณะที่เอื้อต่อการขยายตัวของปรากฏการณ์ลัทธิ (หน้า 60)

### 2. การบังคับใช้กฎหมายวันที่ 9 ธันวาคม 1905 ว่าด้วยการแยกศาสนจักรออกจากรัฐ

- ย้ำให้ชัดเจนผ่านหนังสือเวียนถึงผู้ว่าราชการจังหวัดและหน่วยงานที่เกี่ยวข้องทั้งหมด เกี่ยวกับหลักการแยกศาสนจักรออกจากรัฐ และหลักเกณฑ์ในการเป็นสมาคมทางศาสนาที่กำหนดโดยสภาแห่งรัฐ ตลอดจนตัวอย่างการนำไปใช้ในกรณีของลัทธิพยานพระยะโฮวา (หน้า 71) ;
- เรียกร้องให้ฝ่ายการเมืองคำนึงถึงความจำเป็นที่จะต้องกำหนดกฎเกณฑ์ทั่วไปบางประการสำหรับการเข้าร่วมระบบประกันสังคมของศาสนจักร (หน้า 75)

### 3. การปรับปรุงกฎหมายที่เกี่ยวข้องกับการเงินของชีวิตทางการเมือง

- กำหนดให้การได้รับส่วนแรกของเงินอุดหนุนงบประมาณประจำปีต้องผูกกับการบรรลุคะแนนเสียงขั้นต่ำหนึ่งในครั้งสุดท้ายของการเลือกตั้งสมาชิกสภาผู้แทนราษฎร (หน้า 84) ;
- จำกัดการเข้าถึงการรณรงค์ผ่านสื่อวิทยุโทรทัศน์ไว้เฉพาะพรรคการเมืองที่ได้เสนอชื่อผู้แทนระดับชาติ ซึ่งได้รับการสนับสนุนจากผู้แทนท้องถิ่นจำนวนขั้นต่ำที่กำหนด (หน้า 84)

### 4. การจัดระเบียบภาคธุรกิจด้านสังคมและสุขภาพ

- แจกจ่ายคู่มือปฏิบัติให้แก่หน่วยงานสาธารณะทั้งหมดและผู้ประกอบวิชาชีพด้านสุขภาพ ซึ่งระบุจุดเปราะบางต่ออิทธิพลของลัทธิในโลกทางการแพทย์และสังคม (หน้า 128) ;
- รวมศูนย์ข้อมูลเกี่ยวกับบทลงโทษที่ออกโดยสภาวิชาชีพด้านสุขภาพ และเกี่ยวกับกระบวนการพิจารณาคดีที่กำลังดำเนินอยู่ (หน้า 128) ;
- ให้คณะแพทยสภาแต่งตั้งตัวแทนหนึ่งคนประจำภารกิจระหว่างกระทรวงเพื่อต่อสู้กับลัทธิ (หน้า 128) ;
- ศึกษาแนวทางในการจัดให้มีขั้นตอนรับรองคำเรียกขานของวิชาชีพพาราเมดิคและผู้ช่วยทางการแพทย์ โดยใช้คณะกรรมการผู้เชี่ยวชาญที่มีอำนาจอนุญาตให้ใช้คำนำหน้าหรือชื่อเรียกของวิธีการบำบัด และสามารถเสนอให้สั่งห้ามการปฏิบัติวิธีการที่ไม่ได้รับการรับรอง (หน้า 128) ;
- เสริมสร้างการเรียนการสอนในหลักสูตรแพทย์และพาราเมดิคเกี่ยวกับ:
  - วิธีการช่วยเหลือด้านจิตวิทยาแก่ผู้ป่วย (หน้า 129) ;
  - สถานะ บทบาท และขอบเขตของวิชาชีพแพทย์และพาราเมดิคในสังคม (หน้า 129) ;
- พัฒนาการสอนเรื่องการดูแลสุขภาพ อนามัยชีวิต และความเข้าใจโครงสร้างร่างกาย ตั้งแต่ระดับประถมหรือมัธยมศึกษา (หน้า 129) ;
- จัดวางกระบวนการรับรองหรือให้ฉลากสำหรับสมาคมที่เข้าไปมีบทบาทในการดูแลและให้ความช่วยเหลือผู้ป่วยในโรงพยาบาล (หน้า 129) ;
- วางแนวทางการติดตามรักษาทางจิตใจและสังคมเพื่อเสนอต่ออดีตสาวก และผนวกแนวทางนี้ไว้ในการอบรมต่อเนื่องของแพทย์ (หน้า 129) ;
- ขยายการรณรงค์ให้ความรู้เกี่ยวกับวัคซีน โดยเฉพาะภายในระบบการศึกษาแห่งชาติ (หน้า 129)

### 5. การกำกับดูแลด้านการฝึกอบรมวิชาชีพ

- มอบหมายให้การรับรองและการตรวจสอบองค์การฝึกอบรมวิชาชีพเป็นหน้าที่ขององค์การมหาชนทางปกครองแห่งหนึ่ง ซึ่งจะได้รับจัดสรรส่วนหนึ่งของเงินสมทบจากนายจ้างเพื่อใช้เป็นงบดำเนินงาน (หน้า 158 และ 159) ;
- ยกเลิกระบบการได้รับยกเว้นภาษีมูลค่าเพิ่มโดยอัตโนมัติ และจำกัดการยกเว้นไว้เฉพาะองค์การที่ได้รับการรับรองอย่างถูกต้องเท่านั้น (หน้า 159) ;
- ให้อำนาจฝ่ายปกครองอย่างชัดแจ้งในการเพิกถอนการจดทะเบียนขององค์การฝึกอบรม หากปรากฏว่าองค์การนั้นไม่ได้อยู่ในกรอบของการฝึกอบรมวิชาชีพ หรือเข้าไปเกี่ยวข้องกับการทุจริต (หน้า 159) ;
- เปิดการรณรงค์สร้างความตระหนักรู้ที่มุ่งเป้าไปยังภาคธุรกิจ จัดโดยรัฐและองค์การวิชาชีพ (หน้า 160) ;
- พัฒนาโครงการให้ฉลากรับรององค์การฝึกอบรม (หน้า 161)

### 6. มาตรการเพื่อปรับปรุงการต่อสู้กับการฉ้อโกง

a) การปรับปรุงกระบวนการทางอาญา

- เปิดการหารือถึงความเหมาะสมในการจัดตั้งความผิดฐาน “การชักจูงทางจิตใจ” (หน้า 216) ;
- ศึกษาความเป็นไปได้ในการกลับภาระการพิสูจน์ ในกรณีการโอนเงินที่มีมูลค่าสูงที่สุดเมื่อเกี่ยวข้องกับลัทธิ ทั้งนี้ต้องอยู่ในกรอบการเคารพอย่างเคร่งครัดต่อหลักการสันนิษฐานว่าเป็นผู้บริสุทธิ์ (หน้า 218)

b) การระดมฝ่ายปกครอง

- กำหนดกรอบทั่วไปของการดำเนินการของรัฐ โดยคณะกรรมการระหว่างกระทรวงภายใต้อำนาจของนายกรัฐมนตรี และด้วยความช่วยเหลือทางเทคนิคของภารกิจระหว่างกระทรวงเพื่อต่อสู้กับลัทธิ จากนั้นให้แต่ละกระทรวงออกหนังสือเวียนกำหนดรายละเอียดการดำเนินการของตน (หน้า 249) ;
- จัดให้มีตัวแทนของสถาบันการเงินร่วมอยู่ในภารกิจระหว่างกระทรวงเพื่อต่อสู้กับลัทธิ (หน้า 182) ;
- ศึกษาความเป็นไปได้ในการสร้างโครงสร้างระดับจังหวัดที่ขึ้นต่อภารกิจระหว่างกระทรวงเพื่อต่อสู้กับลัทธิ ซึ่งมีหน้าที่ (หน้า 249):
  - จัดตั้งศูนย์ข้อมูลสำหรับประชาชน ภาคธุรกิจ และสถาบันการเงินในแต่ละจังหวัด ;
  - ให้คำแนะนำและชี้แนะแก่เหยื่อ อดีตสาวก และครอบครัวของพวกเขา ;
  - ให้ข้อมูลที่จำเป็นต่อหน่วยงานที่เกี่ยวข้องเพื่อใช้ในการดำเนินงาน ;
  - รวบรวมข้อมูลทั้งหมดที่มาจากฝ่ายปกครองเกี่ยวกับกิจกรรมของลัทธิในจังหวัดนั้น ;
- พิจารณาความเป็นไปได้ในการตั้งตำแหน่งผู้พิพากษาที่ได้รับการฝึกอบรมเฉพาะด้านเพื่อจัดการประเด็นที่เกี่ยวข้องกับลัทธิในเขตอำนาจของศาลอุทธรณ์แต่ละแห่ง โดยหน้าที่นี้จะเป็นส่วนเสริมที่มีประโยชน์ต่อบทบาทของผู้ประสานงานเรื่องลัทธิที่ปฏิบัติหน้าที่อยู่ที่อัยการในปัจจุบัน (หน้า 250 และ 251) ;
- เปิดการหารือภายในสหภาพยุโรปเพื่อนำไปสู่การจัดทำอนุสัญญาเฉพาะว่าด้วยการต่อสู้กับลัทธิ (หน้า 251) ;
- ดำเนินการทางการทูตในกรอบสภายุโรปในประเด็นอันตรายของลัทธินิยม การมีอยู่ของลัทธิในองค์การพัฒนาเอกชนที่อ้างพันธกิจด้านมนุษยธรรม และความเสี่ยงที่เกิดจากอนุสัญญาปี 1986 (หน้า 92)

### 7. การติดตามผลข้อเสนอ

ขอให้รัฐบาลมอบหมายให้ภารกิจระหว่างกระทรวงเพื่อต่อสู้กับลัทธิจัดเตรียมมาตรการเพื่อให้ข้อเสนอข้างต้นนำไปปฏิบัติได้จริง
